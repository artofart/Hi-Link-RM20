# OpenWRT для Hi‑link RM20 с Podkop и Zapret

Готовая прошивка на основе OpenWRT для маршрутизатора **Hi‑link RM20**

## 📦 Что внутри?

- **OpenWRT** (последняя стабильная версия для RM20)
- Минимально необходимые сетевые утилиты и веб‑интерфейс LuCI

## ⚠️ Перед началом

- Только для **Hi‑link RM20**

## 📥 Загрузка прошивки

Скачайте соответствующий файл из репозитория:

- **factory.bin** – для первичной прошивки с завода  
- **sysupgrade.bin** – для обновления уже установленной OpenWRT

> Файлы лежат в папке `openwrt-25.12/`
