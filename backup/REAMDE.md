# Восстановление роутера через UART (MT7981)

Данная инструкция описывает процесс восстановления загрузчика и прошивки роутера на базе чипа MediaTek MT7981 с использованием **UART** и инструментов `mtk_uartboot` и `PuTTY`. Подходит для случаев, когда роутер не загружается стандартным образом (brick), но UART-консоль доступна.

> ⚠️ **Внимание!** Процедура может аннулировать гарантию устройства и требует осторожности. Неправильные действия способны окончательно вывести роутер из строя. Вы выполняете все действия на свой страх и риск.

## Необходимое оборудование и ПО

- **USB-UART адаптер** (например, CP2102, CH340, FTDI) с поддержкой 3.3V логики.
- **Утилита `mtk_uartboot`** – Скачайте исполняемый файл для Windows.
- **PuTTY** (или любой другой последовательный терминал) – [putty.org](https://www.putty.org/).
- **Образы для восстановления**:
  - `mt7981-ram-ddr3-bl2.bin` – RAM-образ загрузчика BL2 для MT7981 (DDR3).
  - `FIP-HiLink.mtd4.bin` – образ FIP (Firmware Image Package) для вашей модели роутера.


## Подключение UART к роутеру

1. **Разберите роутер** (открутите винты, аккуратно отделите плату).
2. **Найдите UART-контакты**: обычно это 4 пина – **GND, TX, RX, VCC** (VCC подключать не нужно!). Контакты могут быть подписаны или найдены по даташиту/распиновке платы.
   - Распространённые обозначения: `GND`, `TX` (передача с роутера), `RX` (приём на роутер).
3. **Подключите адаптер**:
   - GND адаптера → GND роутера.
   - TX адаптера → RX роутера.
   - RX адаптера → TX роутера.
   - **VCC адаптера не подключайте** – питание подаётся от роутера.
4. Подключите USB-UART адаптер к компьютеру.

## Подготовка компьютера

1. Установите драйвер для адаптера. В диспетчере устройств Windows появится **COM-порт** (например, `COM5`). Запомните его номер.
2. Поместите файлы `mtk_uartboot.exe`, `mt7981-ram-ddr3-bl2.bin` и `FIP-HiLink.mtd4.bin` в одну папку.

## Восстановление роутера

### Шаг 1: Запуск mtk_uartboot

`mtk_uartboot` загружает в оперативную память роутера специальный загрузчик и FIP через UART, обходя повреждённый флеш-накопитель.

Откройте **командную строку (cmd)** в папке с утилитой и выполните:

```cmd
mtk_uartboot -s COM5 --payload mt7981-ram-ddr3-bl2.bin --aarch64 --fip FIP-HiLink.mtd4.bin && putty.exe -serial COM5 -sercfg 115200,8,n,1,N
